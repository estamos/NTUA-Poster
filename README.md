# NTUA-Poster
 LaTeX class for scientific posters with affiliation to the National Technological University of Athens

## Class options

| Paper size |
:-----------:|
A0
A1
A2
A3
A4
A5

| Orientation |
:------------:|
Portrait
Landscape
